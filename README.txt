# E-COMMERCE Store - Product Listing Page using fake API 

## Overview
This project is a simple e-commerce product listing page that fetches data from the [Fake Store API](https://fakestoreapi.com/). The goal is to demonstrate clean front-end development practices using HTML, CSS, JavaScript, and Bootstrap.

## Features
- Responsive layout with Bootstrap grid.
- Display product cards with images, titles, prices, and ratings.
- Loading indicator while fetching data.
- Error handling for failed API requests.

## How to Run Locally
1 -  Clone this repository:
   ```bash
   git clone <repository-url>
   cd <repository-folder>

2 - Open index.html in a browser.


## Challenges
1 - Handling errors gracefully when the API fails.
2 - Ensuring a responsive design across various screen sizes.
